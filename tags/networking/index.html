pivoting Notes on pivoting from HTB Academy’s pivoting module INTRO Lateral movement is not the same pivoting because Lateral Movement is a technique used to gain access to additional hosts within a network environment. This also allows us sometimes to elevate our privileges. However, these are all within the same network. Pivoting is using hosts to cross network boundaries that you usually would not have access to, with the goal being more targeted.
